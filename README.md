# Unmanned Vehicle Telemetry Analyzer (UVTA)

Unmanned Vehicle Telemetry Analyzer (UVTA) is a customizable software platform
designed to visualize, process, and analyze telemetry data from unmanned vehicles.

## Scope
UVTA targets post-mission and real-time analysis for unmanned ground and aerial vehicles.

## Features (planned)
- Telemetry visualization;
- Data processing and analysis;
- Modular and extensible architecture with customizable features.

## Status
Early development / Proof of concept.
